# PHP 数组
单个变量中存储多个值
```php
<?php
$nan=array("hackers","study","hard");
echo "I love " . $nan[0] . ", " . $nan[1] . " and " . $nan[2] . ".";
?>

>> I love hackers, study and hard.
```

# 3.获取数组的长度 - count() 函数
```php
<?php
$test=array("nan","study","hard");
echo count($test);
?>
// 3
```

# 4.遍历数值数组
```php
<?php
$test=array("nan","study","hard");
$arrlength=count($test);
 
for($i=0;$i<$arrlength;$i++)
{
    echo $test[$i];
    echo "<br>";
}
?>
/**
 * nan
 * study
 * hard
 */
```
# 5.关联数组
分配数组指定键  
```php
$grade=array("nan"=>"100","bob"=>"91","tony"=>"51");  //创建分数的关联数组
or
$grade['nan']="100";
$grade['bob']="91";
$grade['tony']="51";
//使用指定键
<?php
$grade=array("nan"=>"100","bob"=>"91","tony"=>"51");
echo "nan的分数是 " . $grade['nan'] ;
?>
```

# 6.遍历关联数组
foreach 本质数组内有一个指针，指向数组元素的第一个元素，它利用移动指针的方式去获取数据  
```php
<?php
$grade=array("nan"=>"100","bob"=>"91","tony"=>"51");
 
foreach($grade as $x=>$x_value)
{
    echo "姓名是" . $x . ", 分数是" . $x_value;
    echo "<br>";
}
?>
```
