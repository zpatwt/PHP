# PHP 循环

# 1.while 循环
重复执行代码块，直到指定的条件不成立    
```php
while (条件)
{
    要执行的代码;
}

<?php
$i=1;
while($i<=5)
{
    echo "num is  " . $i . "<br>";
    $i++;
}
?>
//当i > 5时循环结束
/**
 * num is 1
 * num is 2
 * num is 3
 * num is 4 
 * num is 5
 */
```

# 2.do…while 语句
怎样都执行第一次,条件成立继续，否则结束     
```php
do
{
    要执行的代码;
}
while (条件);

<?php
$i=1;
do
{
    $i++;
    echo "num " . $i . "<br>";
}
while ($i<=5);
?>
// first : num 2
/**
 * num 2
 * num 3
 * num 4
 * num 5
 * num 6
 */
```

# 3.for 循环
事先知道循环次数    
```php
for (初始值; 条件; 增量)
{
    要执行的代码;
}
// 初始值，条件，增量
<?php
for ($i=1; $i<=5; $i++)
{
    echo "num is " . $i . "<br>";
}
?>
/**
 * num is 1
 * num is 2
 * num is 3
 * num is 4
 * num is 5
 */
```

# 4. foreach 循环 适用于数组
```php
// key , value 名称都是可以自由取名的
$x => $x.value
foreach ($array as $key => $value)
{
    要执行代码;
}

<?php
$test=array("nan","study","hard");
foreach ($test as $value)
{
    echo $value . "<br>";
}
?>
/**
 * nan
 * study
 * hard
 */

<?php
$test=array(1=>"nan", 2=>"study", 3=>"hard");
foreach ($test as $key => $value)
{
    echo "key  为 " . $key . "，对应的 value 为 ". $value . "<br>";
}
?>

/**
 * key 为 1，对应的 value 为 nan
 * key 为 2，对应的 value 为 study
 * key 为 2，对应的 value 为 hard
 */
```
