# PHP 函数
内置函数，自定义函数    
一种方法    
类似打包做了一组通用程序，比如打印1-9       
只需要调用函数就能实现echo 1-9      
而不需要再去写里面的代码。

# 1.创建 PHP 函数
```php
<?php
function functionName()
{
    // 要执行的代码
}
?>


<?php
function test()
{
    echo "nan study hard ！！ ";
}
 
test();
?>
// >> nan study hard !!
```

# 2.参数
echo different name     
```php
<?php
function name($names)
{
    echo $names . "<br>";
}
 
echo "My name is ";
name("nan");
echo "My sister's name is ";
name("Bob");
echo "My brother's name is ";
name("Tony");
?>
/**
 * My name is nan 
 * My sister's name is Bob 
 * My brother's name is Tony
 */

// two parameters
// $name , $punctuation
<?php
function name($name,$punctuation)
{
    echo $name  . $punctuation . "<br>";
}
 
echo "My name is ";
name("nan",".");
echo "My sister's name is ";
name("Bob","!");
echo "My brother's name is ";
name("Tony","?");
?>
/**
 * My name is nan.
 * My sister's name is Bob!
 * My brother's name is Tony?
 */
```

# 3.返回值return
```php
<?php
function add($x,$y)
{
    $result=$x+$y;
    return $result;
}
 
echo "1 + 4 = " . add(1,4);
?>
// 1 + 4 = 5
```
